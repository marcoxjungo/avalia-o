principais comandos do INDEX

1. **: Define o tipo de documento e a versão do HTML. 2. **: Elemento raiz que envolve todo o conteúdo da página. 3. **: Contém metadados sobre o documento, como o título e links para scripts e estilos. 4. **: Define o título da página que aparece na aba do navegador. 5. *<meta>*: Define metadados, como o charset (<meta charset="UTF-8">) e a descrição. 6. *<link>*: Vincula arquivos externos, como folhas de estilo CSS. 7. *<style>*: Contém CSS para aplicar estilos à página. 8. *<script>*: Insere ou referencia scripts JavaScript. 9. *<body>*: Contém o conteúdo visível da página. 10. *<h1> a <h6>*: Títulos de diferentes níveis, de h1 (mais importante) a h6 (menos importante). 11. *<p>*: Define um parágrafo de texto. 12. *<a>*: Cria um link para outra página ou recurso (<a href="url">). 13. *<img>*: Insere uma imagem (<img src="url" alt="descrição">). </p> <h>CSS</h> <p>color: Define a cor do texto.</p> <p>background-color: Define a cor de fundo do elemento.</p> <p>font-family: Define a família de fontes.</p> <p>font-size: Define o tamanho da fonte.</p> <p>text-align: Define o alinhamento do texto</p> <p>background-image: Define uma imagem de fundo.</p> </body> </html>
